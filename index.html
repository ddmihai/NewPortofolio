Checkout my work

Grupomania HR

This is a full-stack solution created for Grupomania HR. Here, the employees can create, update, read and delete multimedia posts on the company forum. The web application is secured with Bcrypt and JSON web token, and have several functionalities which allows the admin and the post author to delete and update the posts.

Oh-my Food

This is a Front end static website where I present clients four restaurants from France. The purpose of this project is to become familiar with SCSS, which is a CSS Preprocessor. To enhance the web pages, I added animations to some elements which are triggered by the page reload or by hovering over the specific element.

Fitness Pal

Fitness Pal is a personal project where I learned more about the frontend development with VueJS. For this project backend I used MongoDB and Nodejs which is currently hosted by a free server on Heroku. Fitness Pal is a method that help remember the weights and other gym activities from the antecedent gym visits. Frontend is created with the help of VueJS, and it's design is created with SCSS.

Pekokko

Pekokko is a reviewing website for the lovers of spicy sauces. This website have basic image implementations using multer. The website is secured with JSON web tokens and Bcrypt. The user can create some posts where they can upload images, details of the sauce, can update the sauce, and also like and dislike other users posts.

GoMike web design

GoMike website was my first project where I learned to optimise the SEO of a web page. Mainly, the SEO stands for Search Engine Optimisation, and it's purpose is to allow a large variety of users to access the website. In this project I created a way to make this website accessible and relevant to the Google search engine by optimising the images to an acceptable size, create a relevant web page description, I also took in consideration the SEO guidelines for the minimum font size, and more...

Reservia

Reservia is a website where I present some accommodations located in Marseille. The purpose of this project is to create a static website in order to become familiar with flexbox css. The first part of the website is populated with a searchbar and some filters, second part contain some accommodations, rating and prices, and last part contain the available activities.

Orinoco

Orinoco is an ecommerce website which sells vintage cameras online. The website can store the chosen product in a cart separate page, and then guiding the user to the checkout procedure. The purpose of this project was to introduce the Bootstrap components, grid, and the implementations of my first JavaScript project.
